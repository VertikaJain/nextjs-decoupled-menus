# NextJS Decoupled Menus
Implementing Decoupled Menus using NextJS as frontend due to fast rendering - SSR.

## Technology Stack
1. NextJS - Server Side Rendering
2. Material UI - for TreeView 

## Commands
1. `npx create-next-app project-name`
2. `npm run dev`
